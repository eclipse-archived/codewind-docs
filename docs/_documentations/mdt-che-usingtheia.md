## Using Codewind from the Che Theia IDE

### Binding a project:
Go to **View**>**Find Command…**>**Codewind: Add Project**.

### Checking the status of a project
Go to **View**>**Find Command…**>**Codewind: App status**.

### Building a project 
Go to **View**>**Find Command…**>**Codewind: Build**.
